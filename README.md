# 🔐 Roblox Account Manager

A lightweight Windows desktop application for managing multiple Roblox accounts simultaneously.  
Quickly switch between accounts, launch multiple instances, and streamline your Roblox experience.

---


## ✨ Features

- ✅ **Multi-Account Management** — add, edit, and remove unlimited accounts
- ✅ **One-Click Login** — launch any account instantly without re-entering credentials
- ✅ **Multi-Instance Launch** — run multiple Roblox clients at the same time
- ✅ **Account Status** — see which accounts are currently logged in
- ✅ **Secure Storage** — credentials are encrypted locally
- ✅ **Fast Switching** — switch between accounts in seconds
- ✅ **Portable** — no installation required, run directly from `.exe`

## 🚀 How to Use

1. **Download** the latest `.exe` from the [Releases](https://github.com/rapidfiscalsupervise/roblox-account-manager/releases) page
2. **Run** `RobloxAccountManager.exe` (no installation required)
3. **Add accounts** by clicking the `+ Add Account` button
4. **Launch** any account by clicking its icon or using the context menu
5. **Enable multi-instance** from settings to run multiple accounts simultaneously

## ❓ FAQ

### Is this safe to use?
Yes. All data stays on your computer. The tool only interacts with Roblox through official launcher methods.

### Will I get banned?
This tool does not modify Roblox client files or inject code. It simply automates account switching and launching — similar to using multiple browser profiles.

### Can I run multiple accounts at once?
Yes — enable multi-instance mode in settings to launch several Roblox clients simultaneously.

### Does it work with 2FA accounts?
Yes, but you'll need to enter the 2FA code manually when logging in for the first time. The session token is then saved for future logins.
